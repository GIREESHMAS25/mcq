 Q 7 - Discuss the outcome of the code?
                
                    def func1(n): 
                       if(n==0): 
                          return 0 
                               else: 
                          return(n+func1(n-1)) 
                    def func2(n, result): 
                       if(n==0): 
                          return(result) 
                       else: 
                          return(func2(n-1, n+result))  
                    print(func1(4))
